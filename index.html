How to Install

1
Install .NET 8.0 — Download and run the .NET 8.0 Desktop Runtime (Windows x64). This is required to run NovaEngine.
2
Download NovaEngine — Click the download button above. The zip file will be saved to your Downloads folder.
3
Extract the zip — Right-click the downloaded file and choose "Extract All...". Pick any folder you want.
4
Run the editor — Open the extracted folder and double-click NovaEngine.Editor.exe. The editor will launch.
5
Start creating — Use the editor to build games, browse the store, or import your Steam library.
